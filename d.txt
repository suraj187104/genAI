# 🚀 How to Use Gemini Flash API in Your AI Project

## 1. **Get Your Gemini API Key**
- You need a Google Cloud project with the Gemini API enabled.
- Copy your API key (looks like: `AIzaSy...`).

---

## 2. **Install the Required Python Packages**

```sh
pip install google-generativeai python-dotenv
```
- `google-generativeai` is the official Gemini SDK.
- `python-dotenv` lets you securely load your API key from a `.env` file.

---

## 3. **Create a `.env` File for Your API Key**

In your project directory, create a file named `.env` with:
```
GEMINI_API_KEY=YOUR_API_KEY_HERE
```
**Tip:** On Windows, use Notepad and save as `" .env "` (with quotes) and type as "All Files".

---

## 4. **Sample Python Code for Gemini Flash**

```python
import os
import google.generativeai as genai
from dotenv import load_dotenv

# Load the API key from .env
load_dotenv()
api_key = os.getenv("GEMINI_API_KEY")
genai.configure(api_key=api_key)

def ask_gemini_flash(prompt):
    try:
        # Use the Gemini Flash model (fast, free tier)
        model = genai.GenerativeModel('models/gemini-2.0-flash')
        response = model.generate_content(prompt)
        return response.text
    except Exception as e:
        print("Gemini API error:", e)
        return f"Error: {e}"

# Example usage
if __name__ == "__main__":
    result = ask_gemini_flash("Explain how AI works in a few words.")
    print(result)
```

## 5. **Common Pitfalls & Troubleshooting**

- **404 or model not found:**  
  Use the correct model name: `'models/gemini-2.0-flash'`
- **API key not working:**  
  - Make sure `.env` is in the same directory as your script.
  - Make sure you restart your Python process after editing `.env`.
  - Check for typos in the variable name: `GEMINI_API_KEY`
- **CORS errors (for web apps):**  
  - If using Flask as an API backend, add CORS support with `flask-cors`.
- **Quota or billing issues:**  
  - Free tier has limits. Check your Google Cloud console for usage and errors.

---

## 6. **How Gemini Flash Works**

- You send a prompt (text) to the Gemini API.
- The API returns a generated response (text).
- The Flash model is optimized for speed and cost, making it ideal for most real-time or free-tier projects.

---

## 7. **How to List All Available Models**

If you want to see all models your API key can access:
```python
import google.generativeai as genai
genai.configure(api_key="YOUR_API_KEY")
print([m.name for m in genai.list_models()])
```

---

## 8. **Security Best Practices**

- **Never commit your API key or `.env` file to public repositories.**
- Always load the key from environment variables or a `.env` file, not hardcoded in your code.
